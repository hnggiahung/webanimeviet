🎯 HOW TO USE (DETAILED)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

OPENING THE APP
  → Double-click index.html or start.html
  → Opens in default browser
  → No server needed (static files)

SEARCHING FOR MOVIES
  → Type in search box on homepage
  → Results update in real-time
  → Search by title or description

FILTERING BY CATEGORY
  → Click category buttons (All, Action, Adventure, etc.)
  → View filtered results
  → Active category highlighted

WATCHING A MOVIE
  → Click on movie card
  → Redirects to watch.html?id=[movie_id]
  → Video player loads
  → Episodes list shows on right side

JUMPING TO EPISODES
  → Click any episode in timestamps list
  → Video tua (jumps) to that timestamp
  → Active episode highlights

SAVING WATCHING PROGRESS
  → Watching progress saved every 5 seconds
  → Data stored in browser localStorage
  → When returning: see "Continue from minute X?" notification
  → Click "Xem Tiếp" to resume

CINEMA MODE
  → Click "Rạp Chiếu" button
  → Video goes fullscreen
  → Controls hidden for immersion
  → Press ESC to exit

📝 CUSTOMIZATION
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

CHANGE COLORS
  File: css/style.css
  Look for: :root { }
  Change: --primary-color, --accent-color, etc.
  Example: --primary-color: #6366f1; (Indigo)

CHANGE FONTS
  File: css/style.css
  Look for: font-family
  Change to: 'Inter', 'Roboto', etc.

ADD NEW MOVIES
  File: data/data.json
  Add object with: id, title, description, cover_image, etc.
  Refresh browser

CHANGE SITE TITLE
  File: config.json
  Update: "title": "Your App Name"
  Files: index.html, watch.html (update <title>)

CHANGE BRANDING
  File: index.html, manifest.json
  Update: Logo, colors, social links

  Format: https://www.youtube.com/embed/VIDEO_ID
  Example: https://www.youtube.com/embed/dQw4w9WgXcQ
  Get VIDEO_ID from: youtube.com/watch?v=VIDEO_ID

GOOGLE DRIVE
  Format: https://drive.google.com/file/d/FILE_ID/preview
  Example: https://drive.google.com/file/d/1abc123/preview
  Get FILE_ID from: Share link /d/FILE_ID/

TERABOX / ONEDRIVE
  → Create public share link
  → Copy link to video_url
  → Test that it plays

MOVIES NOT SHOWING
  → Check data/data.json has valid JSON
  → Open DevTools (F12) → Console
  → Look for error messages

VIDEO DOESN'T PLAY
  → Verify video_url format correct
  → YouTube: dQw4w9WgXcQ (no extra stuff)
  → Google Drive: 1abc123... (just the ID)
  → Try different video source

BOOKMARK NOT WORKING
  → Check localStorage enabled
  → DevTools → Application → LocalStorage
  → Try private/incognito window

CSS NOT LOADING
  → Refresh hard: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)
  → Check file path: css/style.css (relative, not /css/)
  → Verify file exists

RESPONSIVE BROKEN
  → Check meta viewport tag present
  → Refresh browser
  → Try different device/browser
